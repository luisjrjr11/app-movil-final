# 📱 Aplicación Móvil Multiplataforma

## 📌 Descripción
Este proyecto consiste en el desarrollo de una aplicación móvil multiplataforma utilizando React Native con Expo. La aplicación fue diseñada como parte de la práctica experimental de la asignatura Aplicaciones Móviles, integrando diferentes componentes fundamentales del desarrollo moderno como interfaces interactivas, conexión con servicios backend, autenticación segura y uso de funcionalidades nativas del dispositivo.

---

## 🚀 Funcionalidades principales

- Registro de usuario con validaciones
- Inicio de sesión mediante autenticación con JSON Web Token (JWT)
- Consumo de API REST desde un backend desarrollado en Node.js
- Uso de geolocalización (GPS)
- Uso de la cámara del dispositivo
- Navegación entre pantallas
- Pruebas unitarias, de integración y End-to-End

---

## 🛠️ Tecnologías utilizadas

### 📱 Frontend
- React Native (Expo)
- TypeScript

### ⚙️ Backend
- Node.js
- Express
- JSON Web Token (JWT)
